# Synent_task5_Superstore_sales_data_Analysis

# 💼 Superstore Sales Data Analysis Project

## 📌 Overview

This project focuses on analyzing **Superstore Sales data** to understand business performance, identify trends, and generate actionable insights.

---

## 🎯 Objective

To evaluate sales performance by analyzing revenue trends, profit distribution, and top-selling products.

---

## 🏢 Internship

This project was completed as part of an internship at **Synent Technologies**, where I gained practical experience in business data analysis and reporting.

---

## 📂 Dataset

* **Name:** Superstore Sales Dataset
* **Source:** Kaggle
* Contains details such as order date, sales, profit, category, sub-category, and product information.

---

## 🛠️ Tasks Performed

### ✅ 1. Monthly Revenue Trends

* Analyzed sales over time
* Created line charts to visualize monthly revenue growth
* Identified peak and low-performing periods

### ✅ 2. Profit Analysis

* Evaluated profit across categories and sub-categories
* Identified high-profit and loss-making segments
* Visualized results using bar charts

### ✅ 3. Top-Selling Products

* Identified top-performing products based on sales
* Displayed top 10 products
* Visualized using bar charts

---

## 🧰 Technologies Used

* Python 🐍
* Pandas
* Matplotlib
* Seaborn

---

## 📊 Output

* Visualizations showing sales trends and performance
* Identification of top-selling products
* Profitability insights across categories
* A clear **business insights report**
